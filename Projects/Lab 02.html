Learning Web Standards: Labs 1 & 2 (H1)

Published on October 24, 2025 by Student Web Developer

My first webpage! At first, the process seemed daunting, with so many tags and structures to remember, but breaking it down step by step made it manageable. Learning how to properly structure an HTML document helped me understand how web pages are built and displayed.

What I Learned (H2)

I quickly learned the necessity of using tags to give content meaning, not just style. The key concepts I absorbed include prioritizing content hierarchy and using sequential heading levels.

Core Semantic Takeaways (H3)

  1. Using semantic structural tags like <header> and <footer>.
  2. Prioritizing content hierarchy using sequential heading levels.
  3. The importance of semantic tags over using elements just for styling.
